# JectpackCompose_MVVM_Hilt_Retrofit_Room
Modern Android App using Jetpack Compose, MVVM, Hilt, Retrofit and Room Database.  This project is a modern Android application built with Kotlin and follows clean architecture principles using MVVM. It uses Jetpack Compose for a fully declarative UI, Hilt for dependency injection, Retrofit for seamless API integration And Room Database for storage
